# Spotify Sentiment Analysis (NLP)

Projekt analizy sentymentu opinii użytkowników Spotify z wykorzystaniem metod NLP i Machine Learning.

## Zakres projektu
- Przetwarzanie danych tekstowych (czyszczenie, tokenizacja, lematyzacja, stopwords)
- Budowa modeli klasyfikacyjnych (Naive Bayes, Logistic Regression)
- Reprezentacja tekstu (BoW, n-gram, TF-IDF)
- Ewaluacja modeli (accuracy, f1-score, confusion matrix)

## Wyniki
Najlepsze wyniki uzyskał model regresji logistycznej z reprezentacją tekstu w postaci n-gram.
